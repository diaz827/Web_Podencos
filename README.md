## 📖 Descripción del Proyecto

Esta página web presenta la información estructurada de forma clara y visualmente atractiva utilizando un diseño estilo "Bento Grid". Los usuarios pueden explorar diferentes secciones sobre el podenco interactuando con las tarjetas, las cuales se amplían para mostrar información detallada sin necesidad de recargar la página.

## ✨ Características Principales

* **Diseño Bento Grid:** Interfaz moderna y ordenada basada en CSS Grid, que distribuye el contenido en tarjetas de diferentes tamaños.
* **Interactividad con Lightbox:** Al hacer clic en las imágenes de los podencos o en las tarjetas informativas, estas se abren en un modal emergente (lightbox) centrado en la pantalla para facilitar la lectura.
* **Video Introductorio Animado:** Cuenta con una pantalla de carga que muestra un vídeo introductorio. Incluye lógica en JavaScript para omitir el autoplay en dispositivos móviles y ahorrar datos/batería.
* **Totalmente Responsivo:** El diseño se adapta perfectamente a pantallas de escritorio, tablets y dispositivos móviles mediante el uso de media queries.
* **Fondo Dinámico:** Utiliza un patrón geométrico de fondo creado con CSS puro que se mueve continuamente, aportando dinamismo visual.

## 📚 Secciones Informativas

La web cubre los siguientes aspectos sobre el Podenco:
* **Introducción y Origen:** Historia desde el antiguo Egipto y su llegada al Mediterráneo.
* **Características Físicas:** Detalles sobre su velocidad, peso, pelaje, etc.
* **Cuidados Básicos:** Guías sobre Alimentación, Salud, Adiestramiento y Convivencia en familia.
* **Variedades:** Enlaces externos a información oficial sobre el Podenco Andaluz, Canario e Ibicenco.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de la página.
* **CSS3:** Estilos, animaciones, diseño Grid y Flexbox.
* **JavaScript (Vanilla):** Lógica para el manejo del Lightbox interactivo, clonación de nodos del DOM y control del video de carga.
